📝 API INTEGRATION:

Endpoint 1: Usage Data
  GET /api/organizations/{orgId}/usage
  Response: 5-hour, 7-day, Sonnet, Opus quotas + reset times
  
Endpoint 2: Subscription Info
  GET /api/bootstrap/{orgId}/app_start
  Response: Subscription tier, capabilities, rate limits

Authentication: HTTP cookies (automatic, no token needed)
